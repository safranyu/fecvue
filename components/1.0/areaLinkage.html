    # areaLinkage 三级地址联动组件

    使用方式：

    1. 在要使用的页面引入该组件 import areaLinkage from '@/assembly/areaLinkage'//组件的位置
    2. 注册组件 components:{ areaLinkage }
    3. 在你要使用该组件的位置挂载该组件 <areaLinkage @getSelectedAddress='getSelectedAddress' />

    参数说明 getSelectedAddress：用于获取子组件传过来的已经选择的地址

    /**
* 获取选择的地址
*/
getSelectedAddress(val){
   var selectedAddress = val;//取到子组件传过来的值
   this.selectedAddress = selectedAddress;
},
