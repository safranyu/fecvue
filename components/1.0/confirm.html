    # Confirm 确认框

    这是删除，提交等需要用户再次确认的提示框的一个组件

    使用方式：

    1. 在要使用的页面引入该组件 import confirm from '@/assembly/confirm'//组件的位置
    2. 注册组件 components:{ confirm }
    3. 在你要使用该组件的位置挂载该组件 <confirm v-show="isAlert" tipMsg='你确定自己是个帅哥吗' contMsg="人送外号，芳心纵火犯" makeSureBtnText="确定" cancelBtnText='取消' boxWidth=250 boxHeight=150 @cancelAction='cancelAction' @makeSureAction='makeSureAction' />

    tips：子组件调用父组件方法，参考文档
